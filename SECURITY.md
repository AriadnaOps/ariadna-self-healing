# Security

## Reporting a vulnerability

If you believe you have found a security vulnerability in this project, please report it responsibly.

- **Do not** open a public issue for security-sensitive findings.
- Contact the maintainers (e.g. via the repository’s “Security” tab or the contact listed in the project) with a description of the issue and steps to reproduce, if possible.
- We will acknowledge your report and work with you to understand and address it. We may request additional details or suggest a coordinated disclosure timeline.

We appreciate your efforts to disclose findings in a responsible manner.
